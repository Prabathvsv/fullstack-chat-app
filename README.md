## üì¶ Installation

1)Clone the repository

2)Navigate to the project directory

3)cd fullstack-chat-app

4)Install dependencies

5)npm install

6)### Setup .env file

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```
7)Run the server

8)npm start
